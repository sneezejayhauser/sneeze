# Code Review Skill

Use this skill when the user wants to review code for quality, security, or performance improvements.

## Available Tools

- `read_skill`: Read other skills for context
- `list_skills`: List available skills
- `run_bash`: Execute shell commands
- `write_file`: Create review reports

## Review Criteria

1. **Correctness**: Does the code do what it's supposed to do?
2. **Security**: Are there any security vulnerabilities?
3. **Performance**: Can the code be optimized?
4. **Maintainability**: Is the code easy to understand and modify?
5. **Best Practices**: Does the code follow language conventions?

## Example Use Cases

- "Review this Python script for bugs"
- "Check this code for security issues"
- "Suggest performance improvements for this function"
