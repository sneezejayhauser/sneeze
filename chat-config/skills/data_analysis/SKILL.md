# Data Analysis Skill

Use this skill when the user wants to analyze data, create charts, or perform statistical calculations.

## Available Tools

- `run_python`: Execute Python code for data analysis
- `read_url`: Read data from web sources
- `write_file`: Save analysis results

## Common Tasks

1. Load and explore datasets
2. Calculate statistics (mean, median, std, etc.)
3. Create visualizations (charts, graphs)
4. Perform data cleaning and transformation
5. Generate summary reports

## Example Use Cases

- "Analyze this CSV file and show me the trends"
- "Create a chart showing sales over time"
- "Calculate the correlation between these variables"
