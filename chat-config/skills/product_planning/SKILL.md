# Product Planning Skill

Use this skill when users want roadmap planning, requirement shaping, or release scoping.

## Framework

- Clarify user goals and constraints
- Define MVP and non-goals
- Break into milestones with dependencies
- Add acceptance criteria and rollout checks

## Deliverables

1. Problem statement
2. Proposed solution and alternatives
3. Milestones with effort/risk notes
4. Launch checklist and post-launch metrics

## Example Use Cases

- "Plan a v1 for team chat"
- "Break this epic into sprint tasks"
- "Define acceptance criteria for this feature"
